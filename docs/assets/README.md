# SiteSeeker Website Assets

This directory contains all the images and assets for the SiteSeeker GitHub Pages website.

## Required Images

Please add the following images to this directory:

### Core Images
- `logo.png` - SiteSeeker logo (32x32px recommended)
- `favicon.png` - Website favicon (16x16px or 32x32px)
- `hero-app.png` - Main app screenshot for hero section (800x600px recommended)
- `og-image.png` - Open Graph image for social media sharing (1200x630px)

### Step-by-Step Images
- `install-step.png` - Screenshot showing installation process
- `permissions-step.png` - Screenshot showing permission requests
- `search-step.png` - Screenshot showing search interface
- `advanced-step.png` - Screenshot showing advanced features

### Developer Image
- `developer-avatar.png` - Developer profile picture (80x80px recommended)

## Image Guidelines

1. **Format**: Use PNG or JPG format
2. **Quality**: High quality, crisp images
3. **Size**: Optimize for web (compress if needed)
4. **Style**: Consistent with macOS design language
5. **Content**: Show the app in action with realistic data

## Recommended Screenshots

### Hero App Screenshot
- Show the main search interface
- Include some search results
- Display the beautiful macOS-native design
- Show both Chrome and Firefox results

### Step Screenshots
1. **Install Step**: DMG file or Applications folder
2. **Permissions Step**: System Preferences > Security & Privacy
3. **Search Step**: Active search with results
4. **Advanced Step**: Bookmark search with 'b:' prefix

## File Naming
- Use lowercase with hyphens
- Keep names descriptive
- Maintain consistent naming convention

## Optimization
- Compress images for faster loading
- Use appropriate dimensions
- Consider retina displays for macOS screenshots
